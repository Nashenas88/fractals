# fractals
This is a small side project for rendering fractals.
It's a rust port of the  "Composing Fractals" paper, available here http://web.cecs.pdx.edu/~mpj/pubs/composing-fractals.pdf.
This renders entirely on the CPU, but does at least use rayon to take advantage of multiple cores.

Note: Requires nightly Rust (because I felt like using `#![feature(type_alias_impl_trait)]`)
